# FiXiY (FInd X In Y)

FiXiY is a lightweight, Java-based desktop utility designed to streamline the process of searching for specific strings within files and folders. Built with a responsive JavaFX GUI, it provides a user-friendly interface for complex searching using regular expressions.

## Features

* **Regex-Powered Search:** Perform advanced pattern matching to locate strings within files and directories.

* **JavaFX GUI:** A modern, intuitive desktop interface for managing search parameters and viewing results.

* **Recursive Search:** Efficiently scan through nested folders to find the data you need.

## Getting Started

### Prerequisites

* **Java Development Kit (JDK) 17 or higher**
* **Maven** (for building the project)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/fixiy.git
cd fixiy
```


2.  Build the project using Maven:
```bash
mvn clean package

```

3. Run the application:
```bash
java -jar target/fixiy-1.0.jar
```

## Usage

1.  Launch the application.
2.  Select the target directory you wish to search.
3.  Enter your search string or regular expression in the input field.
4.  Click "Search" to display the results in the main window.
